# Lineage 3D Web App

An interactive 3D evolutionary simulation running entirely in the browser.

## Quick Start

```bash
cd web
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Test

```bash
npm test
```

## Features
- 3D visualization of evolving organisms
- Real-time statistics and phylogenetic tree
- Parameter controls and preset configurations
- Side-by-side comparison mode
- Python recording playback
- Educational overlays and guided tour

## Tech Stack
- React + TypeScript
- Three.js + React Three Fiber
- Vitest (testing)
